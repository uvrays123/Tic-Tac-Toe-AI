## About The Project

This is an online Tic Tac Toe game built using ReactJS. It uses the Minimax Algorithm to decide its moves as well as uses alpha-beta pruning to find the 
best move given any situation of the game.

### Built With

* ReactJS

<!-- GETTING STARTED -->
## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites

No prerequisites required.

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/uvrays123/Tic-Tac-Toe-AI
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
